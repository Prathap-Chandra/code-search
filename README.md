# code-search

### How to run this:

Make a FireWorks api key on fireworks.ai.

```
cd backend
cp .env.example .env
```

Insert the FIREWORKS_API_KEY

Frontend:

````
cd frontend
npm install
npm run dev
```

Backend"
```
cd backend
python -m venv myenv
source myenv/bin/activate
pip install -r requirements.txt
python app.py
```
````
